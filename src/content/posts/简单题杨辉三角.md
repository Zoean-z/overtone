专门用py写的，主要就两个
一个是二维数组其实就是
```python
a = [[1, 2, 3], [4, 5, 6]]
b = [1, [2, 3], ["a", "b"]]
#引用的时候也是
a[1][2] 
#和java c都一样的
```

第二个也是列表，如果是空列表没办法直接用索引，要用
```
append()
#或者先初始化
list=[0]*5
#这样就能用索引了
```
